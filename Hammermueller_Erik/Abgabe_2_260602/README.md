# Abgabe 2

## Ziel

Automatisierte Erstellung einer Ubuntu VM in Exoscale mittels OpenTofu.

## Verwendete Technologien

- OpenTofu
- Exoscale
- GitHub Actions
- CloudInit
- Ubuntu

## Deployment

Workflow "deploy.yml" ausführen.

## Zerstören der Infrastruktur

Workflow "destroy.yml" ausführen.

## Endpunkte

- /      -> HTML Ansicht
- /api.json   -> JSON API
