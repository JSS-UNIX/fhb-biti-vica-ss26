# FHB BITI VICA SS26 – Abgabe 2: VM Info Webserver

## Zielzustand

Eine öffentlich erreichbare VM auf Exoscale, die unter einer IP-Adresse
technische VM-Informationen bereitstellt:

- **`http://<IP>/`** → HTML-Website mit visuell aufbereiteten VM-Details
- **`http://<IP>/api`** → JSON API mit maschinenlesbaren VM-Details
