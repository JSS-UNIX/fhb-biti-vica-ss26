# BITI VICA 2026

**FH Burgenland BITI Vorlesung, Abgaben Virtualisierung SS26**

## Format

- Erstellen Sie einen Ordner gemäß dem Schema `<Nachname>_<Vorname>`, zB `Drack_Daniel`
- pro Abgabe, erstellen Sie einen Unterordner `Abgabe_<Nummer>_<YYMMDD>`, zB `Abgabe_1_260409`
    - Dokumentationen/Beschreibungen/Erklärungen kommen in eine Markdown Datei gemäß dem Schema `Abgabe_<Nummer>_<Thema>.md`, zB `Abgabe_1_Demo.md`
    - Verwenden Sie **Markdown** - kein pures HTML, AsciiDoc, Plaintext oder andere Formate.
- Etwaige Anhänge wie Bilder etc. können in einem `assets` Ordner unter Ihrem Namensordner abgelegt werden.
- Halten Sie sich bei commit messages an den [Conventional Commits](https://www.conventionalcommits.org/en/v1.0.0/) standard
